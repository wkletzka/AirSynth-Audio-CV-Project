# AirSynth

## 🎵 Overview

This project is an interactive audio-visual instrument that transforms real physical space into a playable instrument. By utilizing a depth camera, the system generates interactive "fields" in the air. Users can literally "play the air," manipulating audio simply by moving their hands through these generated zones.
